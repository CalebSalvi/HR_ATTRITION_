HR ATTRITION 

 # Employee Attrition Prediction & Risk Mitigation

## 📌 Project Overview
Losing high-performing talent is incredibly costly, often ranging from 50% to 200% of an employee's annual salary due to recruitment, onboarding, and lost productivity. This project develops a predictive machine learning pipeline designed to proactively identify "flight-risk" employees, allowing HR departments to implement targeted retention strategies before a resignation occurs.

## 📊 Model Evaluation & Selection
Trained and evaluated multiple architectures, including Random Forest, XGBoost (with SMOTE), and Regularized Logistic Regression.
